# Dvizh UI

Next.js 16 frontend for the Dvizh invite management service.

Runs on **http://localhost:3000**. The API must be running on **http://localhost:5055**.

## Setup

```
npm install
```

Create `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:5055
```

## Running

```
npm run dev
```
