# Lore UI

Next.js frontend for the Lore universe catalog service.

Runs on **http://localhost:3001**. The API must be running on **http://localhost:5045**.

## Setup

```
npm install
```

Create `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:5045
```

## Running

```
npm run dev
```

## Stack

Next.js 16 · React 19 · TypeScript · Tailwind CSS 4 · TanStack Query v5 · React Hook Form + Zod · HeadlessUI · Lucide
